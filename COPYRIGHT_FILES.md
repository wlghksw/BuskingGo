# 저작권 등록용 파일 목록

## 포함할 파일들

### 소스 코드 파일
- 모든 `.php` 파일 (HTML이 포함된 PHP 파일들)
- 모든 `.js` 파일 (JavaScript)
- 모든 `.css` 파일 (스타일시트)
- `.json` 파일 (manifest.json 등)
- `.md` 파일 (문서)

### 프로젝트 구조 파일
- `.htaccess` (Apache 설정)
- `sw.js` (Service Worker)

### 디렉토리 구조
```
BuskingGo-feature/
├── api/              # API 엔드포인트
├── assets/           # CSS, JS 파일
├── board/            # 게시판 페이지
├── config/           # 설정 파일
├── includes/         # 공통 컴포넌트
├── pages/            # 페이지 파일
├── .htaccess         # 웹 서버 설정
├── manifest.json     # PWA 매니페스트
├── sw.js             # Service Worker
├── index.php         # 메인 파일
└── README.md         # 프로젝트 문서
```

## 제외할 파일들

- `.git/` 폴더 (버전 관리 내역)
- `.env` 파일 (환경 변수, 비밀번호 등)
- `node_modules/` (의존성 패키지)
- 로그 파일 (`*.log`)
- 임시 파일

## 참고사항

- 현재 프로젝트는 PHP 기반이지만, 모든 PHP 파일에 HTML이 포함되어 있습니다.
- 저작권 등록 시 소스 코드 전체를 보내시면 됩니다.
- 민감한 정보(DB 비밀번호, API 키 등)가 있다면 제거 후 보내세요.
